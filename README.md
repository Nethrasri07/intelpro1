# intelpro1

Restaurant FAQ Bot
An AI-powered FAQ chatbot for restaurants that answers customer questions about hours, menu, reservations, and more using natural language understanding.

Features
- Natural language understanding for customer queries
- 24/7 automated customer support
- Smart compression with ScaleDown (75% token reduction)
- Easy website integration
- Multi-restaurant support

Tech Stack
- Backend: Python Flask
- AI/NLP: OpenAI GPT + ScaleDown
- Frontend: React/HTML-CSS-JavaScript
- Deployment: Docker
